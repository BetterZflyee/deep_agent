# deep_agent

这里存放的是战略 Agent、市场 Agent 和营销 Agent。欢迎大家通过 Fork + Pull Request 参与贡献。

## 快速开始
- 克隆或 Fork 本仓库后，根据说明添加或修改内容。
- 本项目目前处于早期阶段，欢迎任何改进建议与示例。

## 如何参与
1. Fork 本仓库到你的 GitHub 账户
2. 在你的 Fork 里创建分支：`feature/...` 或 `fix/...`
3. 提交改动并推送你的分支
4. 在你的 Fork 仓库页面点击 `Compare & pull request`，发起 PR 到 `BetterZflyee/deep_agent`

详细流程请参见 `CONTRIBUTING.md`

## 交流与反馈
- 问题反馈：使用仓库的 `Issues`
- 讨论：如果已开启 `Discussions`，欢迎在其中交流
